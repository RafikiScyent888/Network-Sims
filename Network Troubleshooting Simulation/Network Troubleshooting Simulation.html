Network Troubleshooting Simulation

Instructions

A network technician needs to resolve some issues with a customer's SOHO network. The customer reports that some of the PCs are not connecting to the network, while others appear to be working as intended. All devices should be set for VLAN 10.

INSTRUCTIONS

Troubleshoot all the network components. Click on each device to diagnose them. Identify any components with a problem and recommend a solution to correct each problem. Click close on each device when finished.

If at any time you'd like to reset the simulation, click the Reset All button.

Click submit once every device has been viewed and diagnosed.
