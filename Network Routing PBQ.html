Scenario: You are configuring routing on Router R1 in a small enterprise network.

Goal: For each destination network, choose the best next hop or interface R1 should use to reach that network.

Router R1 (You configure this)

Interfaces:

  • G0/0: 192.168.10.1/24 (LAN A)
  • G0/1: 10.0.0.1/30 (Link to R2)
  • G0/2: 10.0.0.5/30 (Link to R3)

Router R2

  • 10.0.0.2/30 (to R1)
  • 172.16.20.1/24 (LAN B)

Router R3

  • 10.0.0.6/30 (to R1)
  • 192.168.30.1/24 (LAN C)
  • 0.0.0.0/0 → ISP (Internet)

1. A host on LAN A (192.168.10.0/24) needs to reach a server on LAN B (172.16.20.0/24). Which next hop or interface should R1 use?

2. A user on LAN A (192.168.10.0/24) needs to access a file share on LAN C (192.168.30.0/24). Which next hop or interface should R1 use?

3. A workstation on LAN A (192.168.10.0/24) is browsing an external website on the Internet. Which route should R1 use to forward this traffic?

Solution & Explanations:
  1. Q1: Correct answer – B) Next hop 10.0.0.2 (R2)
    R1 has a static route to 172.16.20.0/24 via 10.0.0.2, which is R2. Traffic to LAN B should be forwarded to R2.
  2. Q2: Correct answer – B) Next hop 10.0.0.6 (R3)
    R1 has a static route to 192.168.30.0/24 via 10.0.0.6, which is R3. Traffic to LAN C should be forwarded to R3.
  3. Q3: Correct answer – C) Default route 0.0.0.0/0 via 10.0.0.6
    For Internet destinations not in any more-specific route, R1 uses its default route pointing to R3 (10.0.0.6), which then forwards traffic to the ISP.
